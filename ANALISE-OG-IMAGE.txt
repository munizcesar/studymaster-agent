4. COMO DEVERIA SER A IMAGEM IDEAL (PROFISSIONAL)
═════════════════════════════════════════════════════════════════════════════════════

🎨 COMPONENTES ESSENCIAIS:

1. FUNDAÇÃO - GRADIENT BRAND
   ✓ Gradient de 3 cores em diagonal (135 graus):
     - 0%: #5B5BD6 (Azul vibrante)
     - 55%: #7C3AED (Roxo em transição)
     - 100%: #F97316 (Laranja acento)
   ✓ Criar movimento visual diagonal
   ✓ Usar overlay semi-transparente (20-30%) em bege claro para suavizar

2. LOGO/ÍCONE
   ✓ Favicon escalado proporcionalmente (140-160px)
   ✓ Posicionar no canto superior esquerdo
   ✓ Usar o gradient do site como fundo do ícone
   ✓ Elemento visual que se destaque:
     - Ícone de documento com check (simboliza "questões prontas")
     - Cores brancas com accents do brand

3. TIPOGRAFIA HIERÁRQUICA
   ✓ Título Principal:
     * "StudyMaster" em 72-80px, bold, branco
     * Posicionar próximo ao logo

   ✓ Subtítulo (Principal):
     * "Gerador de Questões com IA" em 36-40px
     * Branco com leve brilho (semi-transparente)
     
   ✓ Tagline/Descritiva:
     * "Concursos, ENEM, Material Livre" em 24-26px
     * Cor: Branco com menor opacidade
     * Posicionar abaixo do subtítulo

   ✓ URL/Branding:
     * "studymaster-agent.pages.dev" em 16-18px
     * Canto inferior direito
     * Cor: Branco com baixa opacidade

4. ELEMENTOS DECORATIVOS
   ✓ Círculos/formas blur nas laterais
   ✓ Padrão sutil de fundo
   ✓ Espaçamento harmônico entre elementos
   ✓ Alinhamento geométrico

5. CORES E CONTRASTE
   ✓ Textos em branco puro para máximo contraste
   ✓ Usar shadows/glows sutis para realce
   ✓ Contrast ratio > 7:1 (WCAG AAA)

6. COMPOSIÇÃO LAYOUT
   └─ [LOGO] [TÍTULO + SUBTÍTULO + TAGLINE]
      [Espaço]
      [URL no rodapé]
